🔗 URL del script
Pégala una sola vez y queda guardada en el celular.
📋 ¿Cómo obtener la URL?
1. Abre Google Sheets → Extensiones → Apps Script
2. Pega el código del archivo terrazas_script.gs
3. Implementar → Nueva implementación → App web
4. Ejecutar como: Yo · Acceso: Cualquier persona
5. Copia la URL /exec y pégala arriba
